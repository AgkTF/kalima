3. Border Radius

Balanced scale — three tiers, applied consistently by element type.

Button, Input, Chip, Tab
Button / Input / Chip / Tab
10px
Small interactive elements that benefit from a clean, approachable radius without feeling pill-shaped
Card, Panel
Card / Panel
14px
Entry cards, review cards, search results — slightly softer than buttons for content containers
Sheet, Modal
Sheet / Modal
20px (top corners)
Bottom sheets, side panels, modals — the most pronounced radius signals a distinct surface layer

4. Shadows

Minimal approach — depth comes from warm color separation, not heavy shadows. Two tiers only.

serendipity
The occurrence of fortunate events by chance
literary chance
ephemeral
Lasting for a very short time
temporal
Card (ground)
0 1px 3px rgba(0,0,0,0.04),
0 4px 16px rgba(0,0,0,0.06)
Cards, panels, entry lists — resting on the page

Floating
0 2px 8px rgba(0,0,0,0.06),
0 8px 24px rgba(0,0,0,0.08)
Dropdowns, bottom sheets, modals — elevated above the page
sonder
The realization that every passerby lives a life as vivid as your own
neologism empathy perspective
palimpsest
A manuscript page reused after earlier writing has been erased
etymology history
Shadow is a depth signal, not decoration. Cards rest on the page with a gentle lift. Floating elements (sheets, dropdowns) need more visual separation from dense content beneath them.
